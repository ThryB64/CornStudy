# Documentation — Mais AutoML + Étude Professionnelle

## Document principal

**`docs/00_PROJET_COMPLET_MAIS.md`** est la seule source de vérité du projet.

Il contient absolument tout : vision, données, facteurs, modèles, backtest, roadmap, backlog de tickets, journal d'expériences et critères de fin.

Toutes les décisions, tous les tickets et toutes les expériences futures partent de ce document.

## Archives

Le dossier `docs/archive/` contient les 12 anciens documents de cadrage créés lors de la session de définition du projet (2026-05-09). Ils ne sont plus des références actives. Leur contenu a été intégralement fusionné dans `00_PROJET_COMPLET_MAIS.md`.

## Autres documents

Les autres fichiers présents dans `docs/` (rapports générés, audits) sont des artefacts produits par le code. Ils sont regénérés automatiquement par `make study`.
